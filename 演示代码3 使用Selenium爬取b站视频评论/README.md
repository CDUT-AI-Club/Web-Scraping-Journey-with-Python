# 使用ChromeDriver进行b站视频评论爬取
## 使用流程
- 运行 `./获取cookie.py` 获取cookie文件
- 自定义 `./UP主稿件数据.csv` 中的信息
- 运行 `./b站视频评论爬取.py` 进行爬虫
## 注意事项
- 请到官网 [Chrome for Testing (CfT) 可用性信息中心](https://googlechromelabs.github.io/chrome-for-testing/) 下载你的Chrome对应的ChromeDriver的版本
